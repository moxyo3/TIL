## C++の文法
### 基本
- #include <bits/stdc++.h>とusing namespace std;を最初に書く
- #include <bits./stdc++.h>
C++の機能をすべて読み込む
coutとendlのみの場合は#include <iostream>だけでもOK

- using namespace std;
std::cout, std::endlの"std"を省略可能にする


- main関数からプログラムが開始する
- cout << "文字列" << endl;
- cout << "文字列" << ""文字列"" << endl; 連続で出力もできる
- // /* */ コメント
